Rerank Strategies

Entity MMR
Greedily picks articles that introduce the most new entities (tag/subtopic/sourceLean/title words), weighted by IDF. Co-event neighborhood (cos≥0.25) reduces the diversity bonus for redundant articles.
reranker.py:334-382 _entity_mmr()
Hybrid MMR
Blends entity diversity + semantic diversity: score = α × entity_bonus + (1-α) × embedding_bonus. Balances topic coverage with semantic distinctiveness.
reranker.py:425-481 _hybrid_mmr()
Calibrated
At each step, selects the article that minimizes KL divergence between current cluster distribution and a uniform target, using (sourceLean|subtopic) as the cluster label.
reranker.py:483-543 _calibrated()
Embedding MMR
Greedily picks the article with the greatest semantic distance from already-selected ones (minimizes max cosine similarity), using MiniLM or TF-IDF vectors.
